

## 创建数据库

在 SQLPub 中创建一个新的 Serverless MySQL 数据库非常简单。以下是完整的操作步骤。

### 步骤 1：登录 SQLPub

1. 打开 SQLPub 平台并登录您的账号。
2. 确认账号拥有创建数据库的权限。

### 步骤 2：进入数据库管理页面

1. 在左侧导航栏点击 **数据库管理**。
2. 点击页面右上角的 **新建数据库** 按钮。

### 步骤 3：填写数据库信息

- **数据库名称**：输入您希望创建的数据库名称。
- **版本**：选择 MySQL 版本。
- **区域**：选择部署数据库的区域。
- **计算资源**：根据业务需求选择 CPU/内存配置（Serverless 模式可自动伸缩）。

### 步骤 4：确认与创建

1. 检查填写信息无误后，点击 **创建**。
2. 系统将在几秒钟内部署数据库实例。

### 步骤 5：连接数据库

- 数据库创建完成后，在数据库详情页查看连接信息：
  - 主机地址
  - 端口
  - 用户名
  - 密码
- 使用您的 MySQL 客户端（如 MySQL Workbench、DBeaver 或命令行）连接数据库。

### 注意事项

- 数据库名称必须唯一。
- 确保选择的区域靠近您的应用以降低延迟。
- Serverless 模式会根据负载自动分配资源，但请合理设置最大资源上限以控制成本。

### 小结

通过以上步骤，您可以快速创建并连接 SQLPub 的 Serverless MySQL 数据库，为您的应用提供高性能、弹性扩缩的数据库服务。
