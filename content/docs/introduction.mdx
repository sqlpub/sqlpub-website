## SQLPub 介绍

欢迎使用 SQLPub，一个 Serverless MySQL 平台，让您轻松部署、管理和扩展 MySQL 数据库。

### 核心特性

- **极速启动**：几秒钟即可创建新的数据库实例。
- **弹性扩缩**：自动根据负载调整计算和存储资源。
- **按秒计费**：只为实际使用的资源付费。
- **高可用性**：内置复制与故障恢复机制。
- **安全可靠**：TLS 加密、访问控制及自动备份。

### 快速上手

1. 注册并登录 SQLPub。
2. 创建新的数据库实例。
3. 使用您喜爱的 MySQL 客户端连接数据库。
4. 开始创建表并执行 SQL 查询。

### 使用场景

- SaaS 应用需要可扩展的关系型数据库。
- 数据分析或 BI 工作负载。
- 开发和测试环境快速创建数据库实例。
