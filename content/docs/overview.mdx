## SQLPub 概览

欢迎使用 SQLPub，一个 Serverless MySQL 平台，让您轻松部署和扩展 MySQL 数据库。

### 核心功能

- **极速部署**：几秒钟即可启动新的数据库。
- **无服务器弹性扩缩**：根据工作负载自动调整资源。
- **按秒计费**：仅为实际使用的资源付费。
- **高可用性**：内置复制和故障转移机制。
- **默认安全**：TLS 加密、安全认证和基于角色的访问控制。

### 快速开始

1. 注册并创建新的数据库。
2. 使用您喜欢的 MySQL 客户端进行连接。
3. 立即开始创建表并执行查询。

### 使用场景

- 需要可扩展关系型数据库的 SaaS 应用。
- MySQL 分析型工作负载。
- 需要即时数据库的开发和测试环境。

### 总结

SQLPub 致力于通过提供完全无服务器的体验来简化数据库管理，实现可预测的性能和成本效率。
